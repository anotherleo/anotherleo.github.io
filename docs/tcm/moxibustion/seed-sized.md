# 米粒灸

## 病症

### 肠病湿热蕴结

肠道湿热，在中医学中多归咎于阳明经（大肠、胃）腑实热或湿热蕴结。治疗的关键在于通腑泄热、清利湿热、调和肠胃。

1. 局部选穴 - 调节腹部气机，和肠导滞
   - 太乙
     - 归属：足阳明胃经。
     - 位置：上腹部，脐中上2寸，前正中线旁开2寸。
     - 作用：此穴名就有“调和”之意。能调理胃肠气机，清热利湿。对于腹胀、肠鸣、泄泻、消化不良等胃肠湿热引起的症状效果显著。作为核心局部取穴，直接作用于中焦肠道所在区域，疏通气血郁滞，化解湿热。
   - 大巨
     - 归属：足阳明胃经。
     - 位置：下腹部，脐中下2寸（石门），前正中线旁开2寸。
     - 作用：有调肠胃，固肾气的功效。常用于小腹胀满、肠鸣、腹泻、便秘等症。在腹部穴位中，大巨相对偏下，更侧重下腹部肠道（如下结肠、直肠区域）的气机调节和湿热清除。与太乙配合，覆盖了肠部主要的区域。
2. 远端（循经）选穴 - 清泻阳明经腑之热
   1. 曲池
      - 归属：手阳明大肠经之合穴。
      - 位置：在肘横纹外侧端，屈肘时当尺泽与肱骨外上髁连线中点。
      - 作用：手阳明大肠经合穴（“合治内腑”），具有极强的清热泻火、调和气血、通经活络作用。是清泻阳明热邪（特别是气分实热、湿热）的特效穴和要穴。对于任何阳明热盛或湿热阻滞引起的胃肠问题（肠炎、痢疾、便秘）、高热、咽喉肿痛等，它都是首选远端穴位。直接针对大肠腑病湿热的核心。
   2. 三间
      - 归属：手阳明大肠经之输穴。
      - 位置：微握拳，在食指桡侧缘，第2掌指关节后凹陷中。
      - 作用：五输穴中的“输穴” ，五行属木。输穴的特性之一是“主体重节痛”，但也常用于清热、泻火、通络。作为大肠经输穴，能通利大肠腑气，清泻大肠实热。对于咽喉肿痛、齿痛、热病无汗、目痛、腹胀肠鸣、下利脓血（里急后重）等热邪壅滞大肠的病症有良效。它是曲池清泻阳明腑热的强大助力和补充。
   3. 内庭
      - 归属：足阳明胃经之荥穴。
      - 位置：在足背，当第2、第3趾间缝纹端。
      - 作用：五输穴中的“荥穴” ，五行属水。“荥主身热”，荥穴擅长清泻本经热邪。内庭是清泻胃热、调理胃肠的特效穴。主治齿痛、咽喉肿痛、口㖞、鼻衄、胃痛、腹胀、痢疾、泄泻、足背肿痛等。对于肠道湿热病机中“胃热下迫大肠”的部分以及口渴喜冷饮、口臭、舌苔黄腻等胃热症状尤为有效。清泻阳明经的下游热邪，是这组远端配穴的重要组成部分。
3. 可能的补充：
   - 天枢：大肠经募穴（腑气汇聚处），在脐旁2寸。是调理肠腑、疏调气机、清热利湿的超级核心穴位，常作为治疗各种肠道疾病（包括肠炎）的首选局部穴。非常建议加入或替代太乙/大巨之一作为核心局部取穴。
   - 上巨虚：大肠经下合穴（“合治内腑”）。在小腿外侧，犊鼻下6寸。是治疗大肠腑病的远部要穴，能通腑清热、调和肠胃、止泻通便。
   - 阴陵泉：足太阴脾经合穴。如果湿热兼有脾虚，或湿重于热，需健脾利湿时选用。
   - 足三里：足阳明胃经合穴。具有强大的调理脾胃、补中益气、通腑化湿、扶正祛邪的作用。在热势稍缓或需要兼固正气时使用。湿热壅盛时使用要小心，避免助热。

### 肠胃不适

症状：呕吐、反酸、嗳气、胃痛、胃胀、腹痛、腹泻。无论症状如何表现多样（上吐下泻、气滞胀痛），其根本病机都指向脾胃（中焦）气机运行失调。具体表现为：

1. 胃气不降：本应下降的胃气上逆，导致呕吐、反酸、嗳气。
2. 脾气不升：本应上升运化的脾气郁滞或下陷，导致腹胀（尤其是餐后）、腹泻、部分类型的腹痛。
3. 气机郁滞：气行不畅则痛（胃痛、腹痛）、胀（胃胀、腹胀）。
4. 运化失职：升降失调直接导致消化吸收功能障碍，食物积滞或水湿不化，加重上述所有症状。

选穴：

1. 内关
   - 归属：手厥阴心包经之络穴，八脉交会穴之一，通于阴维脉。
   
   - 位置：腕横纹上2寸，掌长肌腱与桡侧腕屈肌腱之间。
   
   - 关键作用：
   
     - 宽胸理气，和胃降逆：这是内关治疗胃部不适（尤其是上逆症状）的绝对王牌功效。心包经虽然不直接连接胃，但其络脉与三焦相通（心包与三焦相表里）。内关作为络穴，沟通表里经（心包-三焦）；作为八脉交会穴通阴维脉（维系诸阴经，主一身之里），能深层次调节内脏功能，尤其是上中焦气机。 它能有效平复上逆的胃气，对恶心、呕吐、反酸、嗳气疗效卓著。
   - 宁心安神：情绪紧张、焦虑往往是诱发或加重肠胃不适（如肠易激综合征）的重要因素。内关能镇静安神，消除诱因。
     - 双向调节：不仅能止呕，对部分胃肠动力不足（如功能性消化不良的早饱、胀闷）也有一定的调节作用。
   
   - 在黄金三角中的角色：负责解决“上逆”症状（呕、吐、反、嗳）的头号要穴，是调和上中焦气机、降逆止呕的“急先锋”。
   
2. 中脘

   - 归属：任脉，胃之募穴，八会穴之腑会。

   - 位置：前正中线上，脐上4寸，胸剑联合与脐连线中点。

   - 关键作用：

     - 调理中焦，和胃降逆：位于胃脘部，是胃气直接汇聚之所（募穴），也是六腑之气汇聚之处（腑会）。是调理一切脾胃、胃肠疾患的第一要穴、局部主穴。无论是虚是实、是寒是热，只要是中焦脾胃的毛病（痛、胀、呕、泻、痞满、纳呆等），中脘都是基础核心选穴。
   - 化湿降浊：中焦是水湿代谢的中心，中脘能健脾化湿，缓解因湿浊中阻引起的胀、闷、痞、泻。
     - 通调腑气：能促进胃肠蠕动，无论对虚寒性不运还是气滞性不通，都有调理作用。

   - 在黄金三角中的角色：坐镇中焦，统领六腑（特别是胃腑），是调理一切胃肠气机的“中央司令”和“主战场”。 直接作用于病所（胃脘），主导局部气机调畅。
   
3. 足三里

   - 归属：足阳明胃经之合穴，胃之下合穴。

   - 位置：在小腿外侧，犊鼻下3寸，犊鼻与解溪连线上。

   - 关键作用：

     - 健脾和胃，理气降逆：作为胃经的合穴和下合穴（“合治内腑”），足三里是调理脾胃功能的“第一保健要穴”和“远治要穴”。 它强大而全面地调节脾胃功能，既能健运脾气（止泻、增食欲），又能和降胃气（止呕、消胀），还能疏导中焦气机（止痛、除胀）。对于胃痛、腹胀、腹痛、呕吐、嗳气、腹泻、便秘几乎全谱系的消化症状均有卓越效果。
   - 扶正培本：具有显著的强壮作用，能改善胃肠功能，提高免疫力。
     - 双向调节：对胃肠动力、消化液分泌等具有多方面的调节作用（如既可用于消化不良的腹泻，也可用于部分类型的便秘）。

   - 在黄金三角中的角色：巩固后天之本（脾胃），统调全身尤其是下肢气血向中焦汇聚，是全面增强脾胃功能、调畅气机、扶正祛邪的“坚强后盾”和“多面手”。

## 自灸选穴

中医经络学说的精髓之一，就在于“经脉所过，主治所及”和“特定穴系统”的强大代偿能力。通过精心选取胸腹部和四肢的穴位，虽然可能无法100%复制背部俞穴或某些头部穴位的特殊疗效，但完全可以构建一个安全、可行、有效（尤其是针对中焦脾胃病）的自我米粒灸方案，达到调理核心病机的目的，只是疗程可能需要更长、起效更渐进。

核心原则：聚焦中焦与远端强穴。既然目标是替代背/头部穴位以达到类似（尽管可能更慢）的效果，我们需要重点选取：

1. 胸腹部局部穴：直接作用于脏腑（尤其脾胃肠）。
2. 四肢特定穴：尤其是具有强大整体调理功效的合穴、募穴、下合穴、络穴等。
3. 健运脾胃核心穴：作为调理中焦的基础。

针对不同病证的可选替代方案：

1. 替代“背部俞穴”（如脾胃俞、大肠俞、小肠俞、肝俞、胆俞等）
   - 俞穴作用：背俞穴是脏腑之气输注于背腰部的特定穴，直接反映和调理相应脏腑的功能，是诊断和治疗内脏病的重要穴位。不可直接触及。
   - 替代思路：
     - 相应“募穴” + “下合穴”组合：这是最经典的替代俞穴的方法，体现了“俞募配穴”和“合治内腑”原则。
       - 脾胃问题：
         - 胃的募穴：中脘 + 胃的下合穴：足三里
         - 脾的募穴：章门 。注意：章门位于第11肋游离端下缘，自我定位和灸疗可能稍困难且需注意勿灸肋骨边缘。 更常用安全的组合是：中脘 + 天枢(大肠募穴) + 足三里 + 阴陵泉(脾经合穴)。这个组合强健脾胃、化湿利湿的效果显著。
       - 肠腑问题 (大小肠)：
         - 大肠的募穴：天枢 + 大肠的下合穴：上巨虚
         - 小肠的募穴：关元 。 替代方案（尤其针对小肠功能如腹泻、吸收）：天枢 + 关元 + 下巨虚(小肠下合穴)
     - 强效经络合穴：足三里(胃经合穴)、阴陵泉(脾经合穴)本身就是调理脾胃功能的“核武器”，其作用类似甚至超越了单用脾胃俞的效果。
   - 优势：这些穴位均在躯干前部或四肢，易于自我定位和操作。募穴在腑前，下合穴在膝下，合治内腑。
2. 替代“头部穴位”（如百会 — 升提阳气醒脑）
   - 百会作用：常用于气虚下陷（如脱肛、久泻）、清阳不升（头晕、倦怠）。无法自灸。
   - 替代思路：
     - 强化气海、关元：对于气虚下陷型的腹泻、脏器下垂：
       - 气海：在下腹部，脐下1.5寸。为元气之海，能大补元气、固本培元、温阳益气。通过提升一身之气来改善中气下陷。
       - 关元：在下腹部，脐下3寸。为元气所藏之处，能温补下元、益气助阳。效果类似气海且更偏重下焦。用米粒灸温和刺激气海或关元，持之以恒，能缓慢提升整体正气。
     - 重用足三里：足三里除了是胃的下合穴，还是著名的全身性强壮穴、扶正穴。长期温和灸足三里，能补益气血、健运脾胃升清阳，对于因脾虚导致的气陷、神疲乏力有潜移默化的提升作用。
     - 效果对比：替代百会起效会慢得多，更需要坚持。但对于提升整体正气、补益中焦仍有扎实效果。
3. 替代“神志相关穴位”（如背部的心俞、督脉穴）
   - 背景：部分胃肠病与情绪（焦虑、忧郁）相关。背俞穴（心俞、肝俞）常用于调神。
   - 替代思路：
     - 重用手厥阴经调神穴：
       - 内关：不仅是降逆止呕的要穴，还是调畅情志、宁心安神的极重要穴位。它本身就是“络穴”，通阴维脉，能调节内脏和情志。对心肝气郁、焦虑引起的胃肠紊乱效果显著，且容易自灸。
     - 用太冲疏肝解郁：太冲 — 足厥阴肝经的原穴、输穴。是疏肝理气、解郁安神的超级穴位。能有效缓解肝气郁结对脾胃的克犯，改善情绪相关的胃痛、胁胀、嗳气、肠鸣腹泻。位置在足背，可自灸。
     - 效果对比：虽然不如直接灸肝俞、心俞来得“专”，但内关+太冲的组合在调节情志影响胃肠方面依然强大且易于操作。
