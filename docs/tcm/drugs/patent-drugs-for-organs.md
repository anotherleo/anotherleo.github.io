# 脏腑用药（中成药）

## 脾

- 四君子丸
  - 组成：党参、白术、茯苓、甘草。
  - 功效：益气健脾。
  - 适用：脾胃气虚（食欲不振、乏力、便溏）。
- 参苓白术散
  - 组成：人参、茯苓、白术、山药、白扁豆等。
  - 功效：健脾渗湿。
  - 适用：脾虚湿盛（腹泻、肢体困重、舌苔白腻）。
- 补中益气丸
  - 组成：黄芪、党参、白术、当归、升麻等。
  - 功效：补中益气、升阳举陷。
  - 适用：脾虚气陷（内脏下垂、久泻、气短乏力）。
- 归脾丸
  - 组成：黄芪、龙眼肉、白术、当归、酸枣仁等。
  - 功效：健脾养血、安神。
  - 适用：心脾两虚（失眠、心悸、月经量多色淡）。
- 香砂六君丸
  - 组成：木香、砂仁、党参、白术、茯苓等。
  - 功效：健脾化湿、理气和胃。
  - 适用：脾虚气滞（腹胀、嗳气、消化不良）。
- 附子理中丸
  - 组成：附子、干姜、党参、白术、甘草。
  - 功效：温中散寒。
  - 适用：脾胃虚寒（腹痛喜温、畏冷、四肢不温）。
- 保和丸
  - 组成：山楂、神曲、半夏、茯苓、陈皮等。
  - 功效：消食导滞。
  - 适用：脾虚食积（嗳腐吞酸、脘腹胀满）。

### 药物辨析

#### 香砂六君丸 vs. 参苓白术散

| 对比项     | 香砂六君丸                                                  | 参苓白术散                                        |
| ---------- | ----------------------------------------------------------- | ------------------------------------------------- |
| 基础方来源 | 四君子汤（党参、白术、茯苓、甘草） <br/> 木香、砂仁、陈皮、半夏 | 四君子汤（党参、白术、茯苓、甘草） <br/> 山药、莲子、白扁豆、薏苡仁、砂仁、桔梗 |
| 核心药物   | 木香、砂仁（行气化湿）                                      | 山药、薏苡仁（补脾渗湿）                          |
| 主要功效   | 健脾化湿 + 理气和胃                                         | 健脾渗湿 + 益肺固肠                               |
| 兼证调理   | 侧重消胀止呕（气滞症状）                                    | 侧重止泻固肠（湿泻症状）                          |
| 禁忌证   | 胃阴虚（口干舌红）、胃热灼痛禁用      | 湿热泄泻（肛门灼热、大便臭秽）慎用        |
| 联合用药 | 可配保和丸治食积，配柴胡舒肝丸治肝郁  | 可配补中益气丸治脏器下垂，配玉屏风治易感  |
| 疗程     | 饭前服用，一般2-4周（气滞缓解即减量） | 空腹服用，需1-3个月（慢性泄泻需长期调理） |
| 不良反应 | 偶见口干（砂仁温燥）                  | 少数人服薏苡仁后轻微腹泻（可减量）        |

- 适用证型与症状区别
  - 香砂六君丸
    - 核心病机：脾虚 + 气滞湿阻  
    - 典型表现：  
      - 脘腹胀满（饭后加重）、嗳气频繁；  
      - 恶心欲呕、食欲不振；  
      - 大便溏薄但排便不畅（里急后重感[^1]）；  
      - 舌苔白腻、舌体胖大边有齿痕
    - 西医对应：慢性胃炎、功能性消化不良、胃轻瘫。
  - 参苓白术散
    - 核心病机：脾虚 + 湿盛泄泻  
    - 典型表现：  
      - 长期腹泻（水样便或完谷不化）、肠鸣辘辘；  
      - 肢体困重、面色萎黄；  
      - 气短乏力、易感冒（肺脾气虚）；  
      - 舌淡苔白滑、脉濡缓
    - 西医对应：肠易激综合征（腹泻型）、慢性肠炎、营养不良。
- 选择要点：
  - “胀”明显选香砂六君丸（理气优先）；  
  - “泻”明显选参苓白术散（固涩优先）； 
  - 若脾虚兼肺虚（如易感冒、咳痰稀白），参苓白术散更佳。 
- 结合舌脉辨证使用，二者亦可交替使用（如白天香砂六君丸消胀，晚间参苓白术散止泻）。

#### 藿香正气丸

藿香正气丸是中医治疗“外感风寒、内伤湿滞”的经典方剂，主要针对寒湿困阻中焦（脾胃）引起的症状，其核心适应症如下： 

1. 外感风寒 + 内伤湿滞（胃肠型感冒）
   - 症状：恶寒（怕冷）、低热或无热、头痛昏重、胸膈痞闷、恶心呕吐、腹胀腹痛、腹泻（大便稀溏）
   - 病机：外受风寒（如吹空调、淋雨），内伤湿浊（如贪凉饮冷、饮食不洁），导致脾胃升降失常。
   - 适用场景：  
     - 夏季暑湿天气，因贪凉导致的“空调病”
     - 饮食不洁或水土不服引起的急性胃肠炎（非感染性）
2. 湿浊中阻（脾胃湿滞）
   - 症状：舌苔白厚腻、口淡无味、胃脘胀满、食欲不振、嗳气、大便黏滞不爽。  
   - 病机：脾虚运化无力，湿浊停滞中焦，气机不畅。  
   - 适用场景：  
     - 慢性胃炎、功能性消化不良属寒湿证者。  
     - 湿气重导致的头身困重、四肢酸软。  
3. 暑湿泄泻（非感染性腹泻）
   - 症状：腹泻（水样或溏便）、肠鸣、腹部冷痛、小便清长。  
   - 病机：暑季湿邪困脾，寒湿下注肠道。  
   - 适用场景：  
     - 夏季因贪凉或饮食生冷导致的腹泻（无高热、脓血便）。 

##### 藿香正气丸 vs. 香砂六君丸

| 对比项   | 藿香正气丸                                                   | 香砂六君丸                                     |
| -------- | ------------------------------------------------------------ | ---------------------------------------------- |
| 主要成分 | 藿香、紫苏、白芷、陈皮、厚朴、半夏、茯苓、白术、大腹皮、甘草 | 木香、砂仁、党参、白术、茯苓、甘草、陈皮、半夏 |
| 功效侧重 | 解表散寒 + 化湿和中（偏于外感寒湿）                          | 健脾益气 + 化湿行气（偏于脾虚湿滞）            |
| 适应症   | 外感风寒+内伤湿滞（胃肠型感冒）<br>急性寒湿腹泻              | 脾虚湿盛（慢性消化不良）<br>胃胀、嗳气、食欲差 |
| 舌象     | 舌苔白厚腻，舌质淡                                           | 舌淡胖有齿痕，苔白腻                           |
| 适用人群 | 寒湿急性发作期（短期使用）                                   | 脾虚湿滞长期调理（慢性病）                     |

总结：  
- 藿香正气丸适用于外寒+内湿的急性症状（如感冒、急性胃肠炎）。  
- 香砂六君丸适用于脾虚+湿滞的慢性病（如慢性胃炎、消化不良）。  

##### 藿香正气丸 vs. 参苓白术丸

| 对比项   | 藿香正气丸                                     | 参苓白术丸                                       |
| -------- | ---------------------------------------------- | ------------------------------------------------ |
| 主要成分 | 藿香、紫苏、白芷、陈皮、厚朴、半夏、茯苓、白术 | 人参、白术、茯苓、山药、莲子、扁豆、砂仁、薏苡仁 |
| 功效侧重 | 散寒化湿 + 理气和中（偏祛邪）                  | 健脾益气 + 渗湿止泻（偏补虚）                    |
| 适应症   | 寒湿急性症状（呕吐、腹泻、感冒）               | 脾虚湿盛（慢性腹泻、乏力）<br>体弱易感冒         |
| 舌象     | 苔白厚腻（湿浊重）                             | 舌淡苔白腻（气虚明显）                           |
| 适用人群 | 实证（寒湿困脾）                               | 虚证（脾虚湿盛）                                 |

总结：  
- 藿香正气丸用于寒湿实证（急性症状，需散寒化湿）。  
- 参苓白术丸用于脾虚湿盛（慢性病，需健脾渗湿）。 

##### 祛湿药选择

1. 急性寒湿症状（感冒、腹泻） → 藿香正气丸。  
2. 慢性脾虚湿滞（胃胀、消化不良） → 香砂六君丸。  
3. 脾虚湿盛+体弱易疲劳 → 参苓白术丸。  

注意：  
- 藿香正气丸含辛温药（如白芷、紫苏），风热感冒（黄痰、咽痛）禁用。  
- 长期祛湿需结合健脾（如四君子汤、苓桂术甘汤）。

### 使用注意事项

- 辨证用药：脾虚证型复杂，需区分气虚、阳虚、湿盛等。例如：
  - 单纯气虚用四君子丸；
  - 湿重需搭配参苓白术散或香砂六君丸；
  - 阳虚寒象明显选附子理中丸。
- 禁忌与慎用：
  - 补益类（如补中益气丸）：感冒、发热、实热证禁用；
  - 温燥类（如附子理中丸）：阴虚内热（口干、舌红）者忌用；
  - 消导类（如保和丸）：长期脾虚者需配伍健脾药，不可单用。
- 饮食禁忌：
  - 服药期间忌生冷、油腻、辛辣食物，避免加重脾虚或湿滞。
- 特殊人群：
  - 孕妇慎用含附子、枳实等破气药物；
  - 儿童、老人用量需减半或遵医嘱。
- 疗程与联合用药：
  - 健脾药需长期调理（1-3个月），急性症状缓解后仍需巩固；
  - 若兼肝郁（如胁胀）可配逍遥丸，肾阳虚可配右归丸。
- 不良反应：
  - 部分含温热药（如附子）可能引起口干、便秘，需监测；
  - 参类制剂（如人参）可能与西药（如抗凝剂）相互作用。

[^1]: 里急后重(tenesmus)为临床常见症状之一，表现为下腹部不适，很想解大便，然而又无法一泄为快。“里急”形容大便在腹内急迫，窘迫急痛，欲解下为爽；“后重”形容大便至肛门，有重滞欲下不下之感；肛门、直肠及髓尾部坠胀，总有排便不尽感。
