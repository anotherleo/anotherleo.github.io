# 类型学(Typology)

类型学是一个基于荣格心理类型理论的领域，包含了MBTI (Myers-Briggs Type Indicator)，荣格八维 or 毕比模型(Beebe's model)，OPS (Objective Personality System)等不同的理论分支。这个领域对我产生了前所未有的影响，不仅包括自我认识，也包括对他人的认识。为此，我撰写了本文，整理我接触类型学的心路历程，以及对类型学的个人理解。

## 心路历程

我第一次接触类型学是在我大三的时候（2020年），当时隔壁班的cc同学探索到16personalities的测试和《天生不同》(Gifts Differing) 一书，便向我安利。当时我测出来是ENFJ，然后觉得16personalities的描述还感觉挺贴切的，就把这个测试安利给了我身边的人，例如我的舍友们和隔壁宿舍的同学。

身边同学测出来具体是什么类型，我已经忘了。我当时也跟着玩了一段时间，就没再玩下去了，毕竟当时学业也紧张，没什么时间去仔细研究这个东西。

等到我大四的时候（2021年下半年），我去蹭程乐华老师的《心理测量与自我认识》一课。程老师自己提出了一套自己的人格理论，并且强调了这个理论并不是基于心理测量的，因为心理测量很难跳出自己的视角，得到自己没有看到的东西。这套理论相对而言比较简单，只有四个亚型，而且能一定程度上帮到我与其他打交道。当时的我觉得：“MBTI分那么多种，但是用自测的方式，似乎对自己的帮助更小”。加上后来MBTI火起来（我那个时候MBTI还没火起来），很多人开始乱扣刻板印象的帽子，我对MBTI的印象就变坏起来了。以至于有人跟我提MBTI，我就会说道：“这个东西是没有科学依据的，娱乐娱乐就好了，别当真”。

不过，2023年的下半年，因为担任数据结构助教，我跟Œ师兄熟起来了。Œ师兄非常热衷于跟我讲MBTI，而由于此时我对MBTI还是充满负面印象，所以一直给他泼冷水。

事情的转机发生在2024年9月份，当时他跟我分享他在新公司的工作体验时，再一次地提到MBTI。这时的我心态已经有些变化了：“既然这么多人喜欢玩，我为什么不试试看研究研究一下呢？这样还能与朋友们和陌生人多一分谈资。”

但是此时我已经萌发了认真研究一下的冲动了，于是我便打开了bilibili，挑选出几个比较有影响力的MBTI区up主学习一下。

国庆节期间，我继续深入地研究了荣格八维体系。

### 我的日记

- 2024年10月5日

  已经连续地摆了一个星期，莫非是Ne+Se的作用？

  最近这段时间在研究MBTI，然后现在大致确认了自己是INFJ。这次跟之前很不一样的点在于似乎开始发现自己的特别之处也是有原因的。从归因的角度，MBTI确实是有帮助的，因为我现在能对很多我的行为进行归因了。至于这些归因正确与否，还是要看自己能利用这些理论做些什么。

  作为INFJ可能会出现的问题是：

  - Ni
    Ni有时候可能会让我做出不靠谱的决策，不过现在还好，毕竟已经吸取了一定的经验。现在比较大的问题可能是Ni追求一种内在的秩序，而过多复杂的信息会让我找不到秩序，从而产生了虚无和信息过载的感觉。
  - Ne
    当开始做一件事情的时候，可能因为想太多而做不成。
  - Se
    在自己没有什么推进的时候，Se可能会沉迷外界的享乐，比方说疯狂刷B站和知乎。
  - Fe
    太过于注重别人的意见和感受，以至于自己做事情的动机都是来源于他人。

- 2024年10月22日

  当INFJ陷入不好的状态时，Ne总是会跑出来阻碍自己。我为什么那么喜欢刷视频？平时没事的时候刷更多是为了缓解无聊，因为我是一个很害怕无聊的人。但是如果是在状态不好的时候刷，则是忧患意识作怪，想通过获取外界信息来验证自己走的路是否正确。而且我发现如果我陷入到这种状态，和人聊天往往不能解决我的问题，再厉害的人也不行，因为这个状态不是Fi引起的，而是Ne引起的——不是情感和价值观的问题，单纯就是我自己没理好自己想要什么，自己要做什么。当我意志动摇了，就忍不住会去想这些有的没的东西。

- 2024年10月24日

  今天我遇到最大的问题应该是处理各种细节时的力不从心。要读别人写的很杂的代码，自己还要在原来的代码上做修改，而自己的代码已经写得没那么容易修改了。很多事情如果要做的话都得推倒重来，而我现在完全没有积累足够的经验去做。包括我突然需要学习一个新工具，而学习的过程也是很烦，因为总是会有一些因为环境或者文档的问题，只能说工具链一旦定下来之后就不太能经常换了。感觉这些烦心事都是在挑战我的Se。

  看了一些MBTI的视频，感觉这种问题可能是INFJ/INTJ容易出现的问题，因为我的Se比较弱，讨厌细节，而且我的Te也很不好。Te弱导致效率很低，Se弱导致做不了脏活。这两者加起来简直要命。

- 2024年10月26日

  首先还是明确一点：十六型人格指出的是偏好而不是能力。因此，虽然我的Te不好，我的Se也不好，但是能力是可以培养出来的。而且对于INFJ，平衡的关键就是在于发展Se，而我的拖延症很大程度上也是来源于Se不足。

  偏好的要点在于：每个人是通过自己的偏好获取能量和生活的意义，而我的劣势功能Se则妨碍到我获取很多能量。锻炼Se的方法我能想到的主要是正念和运动。

- 2024年11月2日

  这两天主要是看视频很多，其实看知乎还是能有些收获的，但是刷B站和刷小红书就真的是单纯多巴胺驱动，完全停不下来，而且什么都没学到。我自己其实还是抵挡不了关注时事的乐趣，尽管我知道这样浮光掠影其实收获不会很大，主要就是积累一些感性的素材，以便Ni能发挥。其实如果我想看透战略，想看透世界格局的话，深入地学习是必要的。例如，如果我想看透政治，那我就需要对政治机器的运行要有足够的了解——这需要我搜集和记忆大量的素材，包括阅读教科书和相关著作。如果只是单纯通过看一些政论时评，那我是永远隔着一层纱。

  ……

  Ne并不包括在我的自我中。虽然我从发散中得到过很多好处，但是这个功能的使用容易让我失控，让我失去自我，并且陷入负面情绪。因此，处理高密度信息对我来说是把双刃剑——它可能让我学到东西，增加素材，也可能让我失控。使用Ne时需要以使用Ni和Ti为导向：我是为了更加深入地理解某套系统的运作逻辑和事物背后的本质规律才进行发散和搜集资料。如果是毫无目的地发散，则可能给我带来困扰。

- 2025年1月31日

  我的Fe是怎么产生的？可能是因为身边的Fi人，当好几个Fi人同时存在时，Fi的冲突会让我觉得很难受，从而开始摒弃自己的Fi。我的Ni与Fe其实配合得很好，Ni帮我看清冲突的根源，在此基础上我会尝试调用Fe去解决问题。

  Ti的发展可能跟学习科学关系比较大，我感觉在学习数学分析之前，我基本上还是靠Ni在过活。像数理逻辑这种课程对我的影响就很大，因为它把人类思考的底层逻辑公式化出来。

  之所以我发展出Fe-Ni轴，应该是因为困扰我的问题更多是人与人之间的冲突问题。为了解决冲突，我能直接用到的手段就是Fe，而为了理解冲突背后的原因，我发展起自己的Ni和Ti。

  之所以INFJ会如此在意人与人之间的冲突问题，可能跟INFJ本身容易吸收负面情绪关系很大，可能是跟共情能力有关？

  想起柏拉图的观念世界，我觉得这是Ti的典型表现。

- 2025年2月21日

  我想了想，我刷视频/知乎其实不是Ne，其实还是Ni+Se——我想通过Se获取尽可能多的事实来提取一个足够好的基本规律。倒也不是我觉得我自己的Ni总结出来的规律有问题，而是就是单纯想获得一个更好的基本规律。这种应该不算是Ni-Ti loop，因为(1)我没怎么用Ti，(2)获取外界信息本身应该就算是Se。如果是我自己疯狂地写反思，那可能是Ni-Ti loop。当然了，如果是刷到一些无意义会上瘾的东西，那可能就是纯粹在逃避现实，被多巴胺所支配了。

  我之前觉得的我不适合做工程应该不是因为Te的问题，更多是因为Se的问题。我的Te似乎也不是那么弱，虽然完成实验时我不怎么做规划，但是这是因为我想在这个过程中巩固理论知识（优先Ti）。如果我单纯是为了完成一个任务而完成一个任务，我也能运用Te去做。Se弱主要导致了我在完成工作过程中(1)非常烦琐碎的、重复的东西，(2)不是很能体会到一些乐趣。

- 2025年3月31日

  我至今缺乏明确的目标，可能是因为我的几个认知功能之间的互相拉扯形成矛盾所致。例如：

  - Ni与Ti的矛盾：Ti难以容忍Ni得到信息的模糊，Ni难以容忍Ti过于重视细节和局部
  - Ti与Fe的矛盾：Ti追求精确，Fe追求群体氛围和谐，这两者经常形成矛盾

  例如，如果我想一心做学术（Ni、Ti），则Fe会在其中使我犹豫；如果我想改变世界，Ti则会让我从现实世界中退回。或许这其中的原因是因为我的Ti过于弱势，不愿意按照自己的想法去改造世界？还是Fe致使自己对待外界时优先使用情感？
