## 辅音字母

泰语辅音的分类直接决定了音节的基础声调（基调）以及声调符号的实际发音效果：

* 中辅音 (Middle Consonants)：声调表现最为规律。在活音音节中，其声调输出与所标注的声调符号完全一致（即标二调读二调，标三调读三调）。
* 高辅音 (High Consonants)：基础声调为第五调 (Rising Tone)。当该类辅音构成的音节无声调符号且为活音时，默认发升调；其标注声调符号后，读音与符号指示阶位维持一致。
* 低辅音 (Low Consonants)：存在声调平移 (Tone Shift) 现象。该类辅音遵循“发音阶位上移”规律，即标注第二调符号时实际发第三调，标注第三调符号时实际发第四调。

| 辅音类型 | 无符号 (活音) 基调 | 标注第二调符号 (◌่) | 标注第三调符号 (◌้) |
| :--- | :--- | :--- | :--- |
| 中辅音 | 第一调 (中平 / 33) | 第二调 (低平 / 21) | 第三调 (降落 / 41) |
| 高辅音 | 第五调 (上升 / 214) | 第二调 (低平 / 21) | 第三调 (降落 / 41) |
| 低辅音 | 第一调 (中平 / 33) | 第三调 (降落 / 41) | 第四调 (高平 / 45) |
