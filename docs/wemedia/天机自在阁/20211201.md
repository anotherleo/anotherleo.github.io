# 科学命理观：命理学与职业规划

在命理咨询中，人们最常关心的两个项目，就是事业和婚恋。

在事业这一项中，客户最喜欢问我的一个问题又是：八字看我做什么工作比较好？我该往什么方向发展？

写这篇文章是为了给我的客户们提一个醒：在涉及到职业规划时，命理学的意见你应当谨慎的去参考；

更准确的说，命理学的意见应当作为你做决策的一个辅助因素，而不能成为决定性的因素。

**事实上，一个人进行职业规划应当基于三个层面的因素：<br/>一、社会整体的经济发展态势；<br/>二、个人的命运；<br/>三、个人的兴趣和特长；**

**个体是整个社会的一个最小的组成单位，国运的力量、集体的力量是远远大过个人运势的**：

比如同样凶险的一个流年大运的组合，在古代你会的一场重病直接死亡，而在现代你可能只是去医院吃个药、做个手术就解决了；

同样八字的两个人，一个在非洲，一个在中华，这两个人的人生际遇也会截然不同；

因此，我们在去理解“命运”这个概念时，应当清醒的意识到：**个人的努力在社会大势之下是微不足道的**，正所谓天下大势，浩浩荡荡；顺之者昌，逆之者亡。

古代士农工商，当官出仕是最高追求，也是最好的人生出路，所以八字看重官印和得制化的七杀，现代社会，商人和前沿科技的从业者的地位大大提高，所以财星和伤官，特别是伤官的作用得以在现代这个社会得到充分的价值发挥。

而在当前这个全球经济集体呈现出衰退倾向的时代，进入体制内、当程序员进大厂、从事新媒体当网红、搞金融等这几条有限的途径，已经成为事实上绝大多数人择业是最好的选择。

各个传统行业的衰退已经不可避免，如果一个命理师还秉持着僵化的思维，单纯的只按照八字的逻辑去给别人的职业规划提供建议，那么就很可能无意识的引别人走向歧路。

比如有一些非常典型的职业类象的组合：木克土的财破印格，代表的是土木工程行业。

假如说一个算命先生看到一个这样的八字，说兄弟，你这个命就是个干土木工程的料！十年内必定能当上总工，你打灰去吧。

算命先生要是这样讲，我觉得两个月后可能会被客户找上门来揍一顿。

再比如说辛丑日身在寅月，又官杀攻身，这种命的职业取向是小偷，那算命先生总不能看到这种命说让人家专心钻研偷盗技术，成为偷盗界的风云人物吧？

这自然是不可能的。

当然，我觉得大部分的命理师还是具备基本的职业素养的，通常也都是会劝人向善。

但是我同样认为，绝大多数的命理师可能都无法意识到自己专业的局限性。

正所谓拿着锤子，看什么都像是钉子，人们总是会喜欢使用自己熟悉的专业知识的范式、去分析一切。

越是资深的、技术水平高的命理师，越是容易陷入“宿命论”的陷阱，因为他的技术足够高超，令他能够积累大量的算出别人生活中细节的成功经验；

这些成功的经验，会在无意识之中加深他对于命运的强大的认知，进而认为：既然你发生的一切都能够命理中推算出来，那么这就意味着每个人一生的事项都是被安排好了的，无法改变。

去年上半年我进步最快的那段时间也不自觉的加深了宿命论的认知，幸而去年下半年发生了一件事令我幡然醒悟，后面我会再写一篇文章专门论述一下命运中可变和不可变的的文章。

一个命理师习惯了命理学的分析范式，他就会本能的只从命理学的角度来给别人提建议。

但是他们却忽略了，命理学只是用来分析一个人的一套理论，而理论是不等于现实的。

譬如mbti人格测试、大五人格测试等等，这些测试呢，都是从人们常见的共性中，去归纳出一些主要的特点，然后再用这些归纳出来的理论，用在不同的人身上，去分析他们。

但是：任何一种归纳性的理论，势必是不能够精准的对应一个单独的个体的。

哪怕有99%的食神制杀的女命克夫，你也是不能够排除那1%的例外的。

当然，最重要的还是在于：我们做任何事情一定不要忘了自己最根本的目的是什么，不要舍本逐末。

一个人去找命理师咨询自己的事业，他的根本目的必然是为了让自己的事业更够发展的更好、能够选择一个适合他、符合他的需求的事业方向。

如果更直白的讲：人们要选择的事业，应当是能赚钱的、自己能做的、还能做的比较舒服的。

什么事业能赚钱，这一点是社会大势决定的，不是由你个人决定的，因为选择大于努力；

你用同样的力气，在一个夕阳行业一个月就赚个五千，但在一个风口行业，或许能赚到五万。

自己能做什么，这一方面取决于你的学习经验，一方面还要取决于你的天赋，在这一点上，是命理学能够对你起到帮助的。

因为命理学可以像人格分析一样，判断出你的天赋是什么，你大概适合的方向有哪些。

但是在这里我们也要实事求是的承认：天赋也有强弱之分的，你对某件事情有天赋，并不一定意味着你就能够把这件事做得多好。

说句扎心的话，有时候我们说某人在某某方面有天赋，也可能是因为他其他的任何方面都做不好，就这一个方面还具有正常的水平。

最后你做什么比较舒服，这就要看你个人的兴趣和特长所在了。

你喜欢一个东西，喜欢做一件事情，就意味着你做这件事情的时候最起码你不会抗拒，最起码这份工作不会给你带来太多的压力。

一般而言，每个人一生的大部分时间都是花费在工作上的，你的工作倒不是一定条件要多么的优渥，但最起码它不要让你太抗拒，不要带给你额外的痛苦。

这种工作的痛苦我自己是有非常深的体会的，我从大学出来后一共在外面工作过两年半的时间，做过销售和运营，这两年半的时间简直是令我如坐针毡、度日如年，每一分每一秒都想辞职。

最后是实在忍受不了了，我裸辞了，开始做自由职业，开始做我自己感兴趣和擅长的事，这时候我才开始感受到自由，不再因为工作而忍受煎熬。

所以，职业的规划是一件需要考虑多方面因素，来综合做决策的事情，它并不是仅仅只靠命理学这一个分析理论就能够做出最终判断的。

而且你还需要考虑到命理师除了命理学专业技术之外的、其它层面的知识素养，这个命理师对于社会的政治经济大势的理解、对人情世故经历的阅历，这些造就了这个命理师的眼界。

命理师自己的眼界，也决定了他所给你提供的建议的质量和实用性。

有人此时可能会有疑问，这样一来，看起来命理分析的作用似乎很小，和职业性格测试的作用也差不了多少，是不是说用命理学来做职业规划就是无意义的了呢？

当然绝不是这样，命理分析的作用，是性格分析测试无法取代的。

一个人进行择业和职业规划，他首先就应该纯粹的从世俗的角度，结合社会发展的大势和个人的兴趣特长，先框定出几个方向和计划。

事实上我的很多客户，当然也不局限于我的命理咨询的客户，应该说是绝大多数的人，都并没有太认真的去对待自己的职业生涯规划这件事。

很多人都是稀里糊涂的、或者听凭家长帮他选择了一个专业，毕业了就稀里糊涂的找个工作做了几年。

这其实是一种对自己并不负责的态度。

计算机普及之前，人们的信息获取渠道是有限的，由于有效信息的缺乏，这导致我们的生涯发展不够自主，这倒还算是情有可原；

但是计算机普及之后，知识的垄断和信息传递的限制都被打破，一个人在具备了充分的信息获取自由的情况下，他在做人生重大决定的时候、还不先去搜集必要的信息，这就可谓是对自己的不负责。

譬如你在大学报考专业之前，先去网上搜索一下有哪些天坑专业要规避，有哪些是将来的热门专业；从有前景的热门专业中框定几个，然后在求职APP中，看一看这些专业对口岗位给出的平均薪资状况，就这几个简单的动作，就足以帮你规避掉很多的损失。

如果你已经有了大概的方向，那么接下来就是命理分析发挥巨大作用的时候了：

第一，你需要通过命理分析来帮你规避掉不适合你命格的行业；

人选择事业时，应当尽量避开你八字中忌神所代表的的行业，仅这一点的作用就是极大的，职业性格测试可并不能预测到你从事什么行业会倒霉。

而很多的人，他所选择的职业恰恰是他八字中，破格无制的那个字所对应的——这本身就是八字分析职业的一种断法。

而如果你的职业发展走的是这种路线，往往意味着你从事这个职业会非常的不开心，你没有办法在这个领域沉下心去深耕、去发展，最后的结果是你花了几年甚至几十年的时间耗费在一个并不适合你的职业上。

第二，你需要通过命理分析，对你所框定的事业方向再进行一次对照。

因为你已经通过结合社会大势和个人兴趣，确定了几个基本的方向，那么如果再与你八字中喜神和用神所对应的行业，进行交叉对比，就能够为你的选择再加上一道保险。

第三，你需要通过对自己今后运势的分析，来做好职业规划和财务规划。

大运的力量是极其强大的，它很难通过个人的努力去对抗，如果你连续要走十年甚至二十年的忌神运，这通常并不会有太好的破解办法。

因为忌神运不仅影响你的运气，它还会影响你的认知和思维，它会让你认为一些偏激的、错误的、消极的想法是正确的、是天经地义的。

认知和情绪决定了你的行动，你的脑子里装满了错误的认知，自然就会容易总是做出错误的行动。

走忌神运时，一般而言是一动不如一静的，这时候你的目标应该是少犯错，而不是多获得。

如果能够提前知道自己会在哪个时间段、哪个年龄阶段走忌神运，这就可以令你做出更合理、也是更有预见性的职业规划，譬如你41~51岁走忌神运，那么你就应当争取在40岁之前多攒些足够的本钱，用于应对这十年降低的收入；

譬如你知道自己32岁时可能会生一场比较大的病，那你就应该提前做好身体的锻炼，在生病之前一年，多结交同行，一边因为生病而失业后，找不到好的去处。

反过来，你也可以提前预知到你什么时候走好运，这样你就知道自己应该在哪个阶段持续发力，奋发进取。

亦或是通过提前确定了你重要的时间节点，从而早做规划；

比如你今年大三，你发现自己未来三年的考试运都不好，那么你就不妨毕业后先参加一年工作，然后再备考公务员或者考研；

如果你知道自己明年事业运势好，有好的工作机会出现，你自然应当及时的去跳槽。

从命理师的角度，应该思考如何尽可能的给来访者提供帮助；

但是从来访者的角度，你应该学会有意识的区分命理师给你的讲解中，哪些是有效信息，哪些是无效的信息，并尽可能的去选择那些有效信息，来帮助你做出更好的决策。
